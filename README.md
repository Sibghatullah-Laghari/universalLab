# 🧪 Universal Lab Barcode System

A lightweight Java-based application to generate and scan barcodes for laboratory sample tracking. Built with the ZXing library, this project simulates a real-world lab sample management system with a focus on simplicity, scalability, and performance.

---

## 🚀 Features

- ✅ Generate barcode images for lab samples
- ✅ Decode barcodes from image files
- ✅ Sample data includes patient info, test type, and date
- ✅ Fast, portable, and easy to customize
- ✅ Designed to be scalable and client-ready

---

## 💻 Technologies Used

- **Java 17**
- **Maven**
- **ZXing (Zebra Crossing) Library**
- **IntelliJ IDEA**
- (Optional Future: MySQL, JavaFX, Swing, REST API)
